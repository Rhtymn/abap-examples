# ABAP Examples

A collection of ABAP code examples that accompany my blog tutorials. Each folder contains complete, working source code that you can use as a reference or adapt for your own projects.

## Examples

| Folder | Description |
|--------|-------------|
| [spta-framework](./spta-framework) | Sales Order Report using SPTA Framework for parallel processing |

## About

These examples are companion code for my blog posts. Each example is a self-contained ABAP program that demonstrates a specific concept or technique in SAP ABAP development.

## How to Use

1. Browse the folders to find the example you need
2. Copy the source code into your SAP system via SE38 or ADT (Eclipse)
3. Adjust the program name and any project-specific references as needed
